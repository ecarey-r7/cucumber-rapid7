# Cucumber::Rapid7

Cucumber hooks, formatters, and more used at Rapid7 by the ControlsInsight team.

## Installation

Add this line to your application's Gemfile:

    gem 'cucumber-rapid7'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cucumber-rapid7
